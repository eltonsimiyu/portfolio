# 🌐 Elton Simiyu — Developer Portfolio

Welcome to my personal portfolio — a full-stack project showcasing who I am, what I do, and how I build tech with purpose.

## 🔗 Live Site

👉 [https://eltonsimiyu.github.io/portfolio/](https://eltonsimiyu.github.io/portfolio/)

## 🛠️ Technologies Used

- **Frontend**: React.js, Bootstrap, AOS
- **Backend**: Node.js, Express
- **Database**: MySQL (Hosted on Railway)
- **Email Service**: Nodemailer with Gmail App Password
- **Deployment**:
  - Frontend → GitHub Pages
  - Backend → Railway

## 🚀 Features

- Dynamic About Me, Skills, and Experience pulled from database
- Contact Form that stores messages and sends real emails
- Mobile responsive layout and Bootstrap styling
- CSV download for skillset
- CV modal and download
